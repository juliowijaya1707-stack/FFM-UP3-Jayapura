Selamat Datang, Petugas!

Aplikasi ini bantu kamu absen, lapor kerjaan, dan kirim lokasi GPS ke supervisor — otomatis, tanpa perlu foto & WA manual lagi.

🔄

Begini Alurnya

  1. Pilih nama regu & Unit Kerja
  2. Foto selfie sebagai bukti mulai kerja
  3. Pilih status: Standby / Perjalanan / Aktif Bekerja
  4. Tiap selesai 1 titik kerjaan → kirim laporan + foto sebelum/sesudah
  5. Sudah selesai semua? Tekan "Selesai Bertugas"
⚠️

Biar Lancar, Perhatikan Ini

  • Nanti muncul minta izin Lokasi & Baterai — pilih "Izinkan" semua, supaya status kamu tidak "Offline" sendiri.
  • Jangan tutup paksa aplikasi (swipe dari recent apps) selama masih bertugas.
  • Sinyal hilang di lapangan? Tenang — data tersimpan otomatis di HP & terkirim sendiri begitu sinyal balik.

🔋 Supaya Tidak "Offline" Sendiri

HP Android tertentu (Samsung, Xiaomi, Oppo, Vivo, Huawei) punya pengaturan hemat baterai tambahan di luar pengaturan baterai standar Android. Ini yang sering diam-diam "membekukan" aplikasi kalau layar dikunci lama, walau izin baterai standar sudah kamu setujui. Ikuti langkah sesuai merek HP-mu:
Langkah 1 — Battery App Settings
Settings → Apps → (nama aplikasi ini) → Battery → pilih "Unrestricted"
Jangan pilih "Optimized"/"Dioptimalkan" — itu yang bikin GPS mati saat dikunci.
Langkah 2 — Hapus dari daftar "Sleeping Apps"
Settings → Battery and device care → Battery → Background usage limits
Pastikan aplikasi ini TIDAK ada di daftar "Sleeping apps" atau "Deep sleeping apps". Kalau ada, hapus.
Langkah 3 (kalau masih bermasalah) — Matikan fitur global
Settings → Battery and device care → Battery → titik tiga (⋮) → Settings → matikan "Put unused apps to sleep" / "Auto disable unused apps"
Langkah 1 — Battery saver
Settings → Apps → Manage apps → (nama aplikasi ini) → Battery saver → pilih "No restrictions"
Langkah 2 — Autostart
Settings → Apps → Permissions → Autostart → aktifkan untuk aplikasi ini
Tanpa ini, MIUI bisa mematikan app begitu layar dikunci.
Langkah 3 — Lock dari recent apps Buka daftar recent apps, tahan kartu aplikasi ini, lalu tap ikon 🔒 gembok supaya tidak ikut ter-swipe/dibersihkan RAM cleaner.
Langkah 1 — Battery Optimization
Settings → Battery → App Battery Management → (nama aplikasi ini) → pilih "Allow background activity" / "Don't optimize"
Langkah 2 — Startup Manager
Settings → App Management → App Startup Manager → aktifkan izin start otomatis untuk aplikasi ini
Langkah 3 — Lock dari recent apps Sama seperti Xiaomi, tahan kartu app di recent apps lalu kunci (🔒).
Langkah 1 — Background power consumption
Settings → Battery → Background power consumption management → (nama aplikasi ini) → "Allow"
Langkah 2 — High background power consumption
i Manager → App manager → Autostart → aktifkan untuk aplikasi ini
Langkah 1 — Launch (App launch management)
Settings → Battery → App launch → cari aplikasi ini → matikan "Manage automatically", lalu aktifkan Auto-launch, Secondary launch, dan Run in background
Langkah 2 — Lock dari recent apps Tahan kartu aplikasi di recent apps lalu kunci (🔒) supaya tidak dibersihkan sistem.
Langkah umum untuk semua HP Android
Settings → Apps → (nama aplikasi ini) → Battery → pilih "Unrestricted" / "Tidak Terbatas" / "No restrictions"
Cari juga menu-menu ini kalau ada "Autostart", "Background app management", "Power saving whitelist", atau "Protected apps" — aktifkan untuk aplikasi ini.
Tombol di atas mencoba lompat langsung ke layar pengaturan yang relevan. Kalau HP kamu tidak mendukung lompatan otomatis, ikuti langkah manual di atas.

🛰️ Setup Awal Sekali Saja

Supaya lokasi & kamera kamu berfungsi normal seharian, selesaikan 3 langkah di bawah. Cuma perlu dilakukan sekali — setelah ini tidak akan diminta lagi.
① Izin Lokasi "Sepanjang Waktu"
Belum dicek.
Setelan HP → Aplikasi → NUMBAY Petugas → Izin → Lokasi → pilih "Izinkan sepanjang waktu"
② Izin Kamera
Belum dicek.
Setelan HP → Aplikasi → NUMBAY Petugas → Izin → Kamera → pilih "Izinkan"
③ Matikan Pembatasan Baterai
Belum dicek.
✅ Semua siap! GPS & kamera kamu akan berfungsi normal seharian.
